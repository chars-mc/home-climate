# HOME CLIMATE WITH ARDUINO
Internet of things project to control the climate and lights in a house.

## Sensors and modules
* Arduino Mega 2560
* Bluetooth HC-06
* DHT11
* FC-28
* LDR
* PIR
* Passive buzzer
